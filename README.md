# 📚 Book Store Website

A full-stack Book Store web application with separate frontend and backend. Users can browse books and access product links, while the backend handles data and APIs.

---

## 🚀 Features

- 📖 Browse books/products
- 🔗 Direct link to product pages
- 🧑‍💻 Full-stack (Frontend + Backend)
- 📦 Organized project structure

---

## 🛠️ Tech Stack

### Frontend
- HTML, CSS, JavaScript, TailwindCss
- React.js , Motion, Redux 

### Backend
- Node.js
- Express.js
- MongoDB

---

## 📂 Project Structure

```

book/
├── frontend/
└── backend/

````

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/rohitsingh0318/Book-Store-website.git
cd Book-Store-website
````

### 2. Setup Backend

```bash
cd backend
npm install
npm start
```

### 3. Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
